# Online Blood Bank Management System

## Overview

The Online Blood Bank Management System is an Android application designed to simplify blood donor management and blood request handling. It allows users to search for blood donors based on blood group and location while enabling administrators to manage donor records efficiently.

## Features

- User Registration
- Blood Donor Registration
- Search Donors by Blood Group
- Admin Panel
- Blood Availability
- Emergency Notifications
- Google Maps Navigation

## Technologies Used

- Java
- Android Studio
- SQLite
- XML

## Software Requirements

- Android Studio
- Java JDK
- Android SDK
