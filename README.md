# 🩸 Online Blood Bank Management System

## 📌 Overview

The **Online Blood Bank Management System** is an Android-based application developed to simplify the process of blood donation and blood requests. The system helps connect blood donors, recipients, hospitals, and administrators through a centralized platform, reducing the time required to find suitable blood donors during emergencies.

This project was developed as a mini project for the Bachelor of Technology (Computer Science and Engineering) program.

---

## 🎯 Objectives

- Provide quick access to blood donor information.
- Reduce the time required to search for compatible blood donors.
- Maintain an organized database of blood donors.
- Improve communication between donors, recipients, and administrators.
- Support emergency blood requests efficiently.

---

## ✨ Features

- 👤 User Registration

- 🏥 Blood Bank Information
- 📍 Donor Search by Location
- 👨‍💼 Administrator Management
- 📱 Android Mobile Application
- 🔔 Emergency Notifications
- 🗂️ Donor Database Management

---

## 👥 User Modules

### Donor
- Register personal details
- Update blood group information
- Become available for blood donation

### Recipient (Acceptor)
- Register into the application
- Search donors based on blood group
- View donor details

### Administrator
- Manage donor records
- Maintain blood bank information
- Monitor users
- Update system information

# 🚀 Installation

1. Install Android Studio.
2. Clone this repository.

```bash
git clone https://github.com/yourusername/Blood-Bank-Management-System.git
```

3. Open the project in Android Studio.
4. Build the project.
5. Run the application on an Android emulator or physical Android device.

---

## 📱 How the System Works

1. The user installs the application.
2. New users register their account.
3. Donors register their blood group and personal details.
4. Recipients search for donors using blood group and location.
5. The administrator manages donor information and system records.
6. The application displays available donors for quick communication during emergencies.
